# 🚀 Yudico Launcher - Oficial

O **Yudico Launcher** é o inicializador exclusivo e personalizado para o servidor **Yudico Live**. Baseado no Helios Launcher, ele foi otimizado para oferecer a melhor experiência de conexão, garantindo que todos os jogadores utilizem a versão correta do jogo e dos mods necessários.

## ✨ Funcionalidades Principais

* **Sincronização Automática:** O launcher gerencia a instalação da versão **1.21.1** e dos mods essenciais.
* **Tradução Completa:** Interface e termos técnicos (como *Marcos de Teletransporte*) totalmente em Português.
* **Segurança:** Sistema de verificação de arquivos para evitar erros de conexão e garantir a integridade dos mods.
* **Integração com Discord:** Status rico mostrando que você está jogando no servidor Yudico.

## 📦 Mods Inclusos

Para garantir a imersão e a comunicação no servidor, o launcher instala automaticamente:
1.  **Simple Voice Chat:** Comunicação por proximidade dentro do jogo.
2.  **JourneyMap:** Sistema de navegação otimizado (com restrições de exploração para manter o desafio).

## 🛠️ Como Instalar e Jogar

1.  Acesse a aba [Releases](https://github.com/maycoveio/yudico-launcher/releases).
2.  Baixe o arquivo `Yudico-Launcher-Setup-2.2.1.exe`.
3.  Execute o instalador (caso o Windows mostre o alerta de "Editor Desconhecido", clique em *Mais informações* > *Executar mesmo assim*).
4.  Faça login com sua conta e clique em **Jogar**.

## 💻 Desenvolvimento

Este projeto é mantido por **Yudico** e faz parte do ecossistema **yudico_live**.

* **Framework:** Electron
* **Base:** Helios Launcher
* **Versão Alvo:** Minecraft 1.21.1 (Paper/Fabric)
